API

        Om tussen verschillende apps en software-onderdelen te communiceren worden API’s gebruikt. Die verbinden de logische laag en de toepassingslaag. 
        API staat voor Application Programming Interfaces. Apps en sites werken vergelijkbaar via de API om data op te halen, bijvoorbeeld van het internet. 
        Zo blijft er consistentie in het apparaat.
